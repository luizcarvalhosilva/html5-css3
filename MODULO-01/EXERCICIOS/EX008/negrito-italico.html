Principais formatações

Negrito / Destaque

Nesta frase usamos negrito de forma não semantica, usando a tag "b".

Nesta frase usamos o negrito de forma semântica com a tag "strong".

Itálico / Ênfase

Nesta frase usaremos o itálico de forma não semântica com a tag i.

Nesta frase usaremos o itálico de forma semântica com a tag "em"

Texto marcado

Nesta frase marcaremos um texto usando a tag "mark"

Também podemos marcar o texto com uma ourta cor usando a css "style" dentro da tag "mark"

Texto grande / Texto pequeno

Podemos usar um texto um pouco maior, com a tag obsoleta "BIG"

Podemos também colocar um texto menor, usando a tag "SMALL" que ainda está em uso

Texto deletado

Podemos marcar que um texto está presente, porém não deve ser considerado.

Texto inserido

Podemos colocar um texto inserido para mostrar que o mesmo foi adicionado depois.

Texto sobrescrito

Podemos inserir texto "elevados" como por exemplo: x2. Com a tag "SUP".

Texto subescrito

Podemos colocar textos em um nível mais baixo como por exemplo: H2O. Com a tag "SUB".
