Posso usar qualquer imagem?

É possível utilizar as imagens do google sem risco de copyright,
porém, é necessário usar filtro("direitos de uso") na busca do navegador.

Opções do Filtro

Sem filtro de licença: será mostrada qualquer imagem, com © ou não

Marcadas para reutilização com modificação: maior nível de direito de uso que é possível ter, pode-se usar a imagem
e ainda modifica-la
Marcadas para reutilização: pode ser usada como bem queira, porém, não podemos modificá-la.
Marcadas para reutilização não comercial com modificação ou sem: não podem ser utilizadas em projetos para faturar com eles
podem ou não ser modificadas.
